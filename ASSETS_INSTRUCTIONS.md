# Instrucciones para Assets

La aplicación necesita algunos archivos de imágenes para funcionar correctamente. Puedes crear estos assets o usar placeholders.

## Assets Necesarios

Crea una carpeta `assets` en la raíz del proyecto con los siguientes archivos:

### 1. `assets/icon.png`
- Tamaño: 1024x1024 px
- Formato: PNG
- Icono de la aplicación

### 2. `assets/splash.png`
- Tamaño: 1242x2436 px (o proporcional)
- Formato: PNG
- Imagen de splash screen

### 3. `assets/adaptive-icon.png`
- Tamaño: 1024x1024 px
- Formato: PNG
- Icono adaptativo para Android

### 4. `assets/favicon.png`
- Tamaño: 48x48 px
- Formato: PNG
- Favicon para web

## Solución Rápida

Si no tienes los assets listos, puedes usar imágenes temporales o crear placeholders simples. La aplicación funcionará sin estos assets, pero es recomendable tenerlos para una mejor experiencia.

## Generar Assets Automáticamente

Puedes usar herramientas online como:
- https://www.appicon.co/
- https://makeappicon.com/
- https://icon.kitchen/

O crear imágenes simples con cualquier editor de imágenes.
